Welcome

This website was built for class (COMP214) Interactive Technologies.


Requirements were to create a 5 pages website with frames, including a permanent header, navigation menu (always displayed, here I added a functionality to close the sidebar with an animated icon) and a third frame into which the sub-pages load.


I have also added a theme customizer (cog icon in middle right), which allows user to toggle dark mode and change header color.


Technologies used as front-end:

  • Angular 10
  • Material Design Bootstrap (MDB)
  • Angular Material

Technologies used as back-end:

  • ASP.NET Core Web Api 5.0

Hosted through IIS on Windows Server 2019 Essentials


Tested and displaying as expected on screen resolutions:

  • 3440 x 1440
  • 1920 x 1080
